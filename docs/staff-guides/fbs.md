# FBS
This is one of the core parts of your job at TechSpark. People will need tools to work, they will need machines to create, and they will need materials to construct. The first thing you do on your shift is log into [FBS](https://cmu.fbs.io/Apps/FBS/), check for any new prints and process them, then begin to assist students.

## 3D Printing Order Processing
1. On the FBS opening page, select the “ORDERING” tab.

![Image](../assets/FBS-1.png)

2. Select “View Orders”

![Image](../assets/FBS-2.png)

3. Filter to “Open,” “Submitted,” and “In-Processing” and select an order number for an Open order.

![Image](../assets/FBS-3.png)
